Welcome to gcs-dev!

About

This repository hosts digital versions of Die Griechischen Christlichen Schriftsteller (GCS) by Adolf Harnack and Theodor Mommsen.

These XML files were produced by data entry company Digital Divide Data (DDD), who corrected and encoded hOCR output generated by Bruce Robertson in accordance with the latest EpiDoc standards. While adequate, more can and should be done to improve the XML (the -dev(elopment) suffix indicates the repository is open to improvement).

Find little more information about GCS on Wikipedia.
